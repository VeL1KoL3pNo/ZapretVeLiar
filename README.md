[--
Update Logs (всегда знайте вышло ли обновление на zapret или veliar)
Launcher (Запускайте и выключайте запрет и выбирайте любой батник)
Network [В РАСМОТРЕНИИ] (Проверьте подключение к ютубу, дискорду в случии чего поменять батник)
Profiles (Сделай профиль с батником и то что он точно работает к тому или иному сайту)
Zapret Settings (Добавляйти адреса сайтов в запрет если у вас его нету и тогда сайт заработал (есть собственный поиск адресов)
Diagnostics (Проверьте ваш запрет на работоспособность)
Statistics (по рофлу статистика)
Plugins (Создавайте личные плагины для veliar'а и публикуйте их на гитхабе)
Logs / Console (Логи: посмотрите есть ли ошибки при запуске или других функций чтобы написать в Issues. Консоль: Консоль для комманд)
Settings (Настройка приложения)
